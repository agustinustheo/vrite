Using provided endpoints, you can retrieve and configure all available **Vrite settings** - the same ones that are configurable in [the settings panel](/settings) - such as your theme or [Prettier configuration](https://prettier.io/docs/en/options.html).

## Retrieve All Settings

Retrieves all settings set in your Vrite account.

<EndpointCard
  client:only
  method="GET"
  endpoint="/settings"
  responseExamples={[
    {
      data: {
        uiTheme: "dark",
        codeEditorTheme: "dark",
        prettierConfig: {
          tabWidth: 4,
          semi: true,
          singleQuote: true,
          trailingComma: "all",
          arrowParens: "always"
        }
      },
      name: "Success",
      id: "success"
    }
  ]}
/>

## Configure Settings

Configures provided settings.

<EndpointCard
  client:only
  method="PUT"
  endpoint="/settings"
  body={{
    schema: "/settings/schema",
    example: {
      uiTheme: "dark",
      codeEditorTheme: "dark",
      prettierConfig: {
        tabWidth: 4,
        semi: true,
        singleQuote: true,
        trailingComma: "all",
        arrowParens: "always"
      }
    }
  }}
  responseExamples={[
    { data: {}, name: "Success", id: "success" },
    {
      data: {
        message: "Input validation failed",
        code: "BAD_REQUEST",
        issues: [
          {
            code: "invalid_type",
            expected: "number",
            received: "string",
            path: ["prettierConfig", "tabWidth"],
            message: "Expected number, received string"
          }
        ]
      },
      name: "bad request",
      id: "bad-request"
    }
  ]}
/>
